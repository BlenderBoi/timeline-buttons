# Timeline Buttons
Add Buttons From Timeline Editor to Other Animation Related Editor

The Play Button, Pause Button, Autokey Options, and Frame Range are Added

Affected Editors
- Sequencer Editor
- Dopesheet Editor
- Graph Editor
- Driver Editor
- NLA Editor
